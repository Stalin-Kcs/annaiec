# Annai Energy & Environmental Consultant

A modern, high-performance corporate website for **Annai Energy & Environmental Consultant**, built with **Angular (v19)** and **Bootstrap 5**. This platform is designed to showcase premium consulting services in energy, environment, and infrastructure management with a focus on sustainability and operational excellence.

---

## 🚀 Key Features

- **Modern Hero Section:** Professional corporate design with high-impact visuals and engaging CTA elements.
- **Mosaic About Us:** Visually stunning grid layout showcasing 30+ years of industry expertise and core values.
- **Mission & Success Tracking:** Interactive counters and success milestones integrated with AOS (Animate On Scroll) for dynamic entry.
- **Comprehensive Infrastructure Solutions:** Editorial-style service cards (Blog Style) highlighting Energy Optimization, Civil Repairs, Water Management, and more.
- **Premium CTA:** A high-conversion "Get In Touch" section with cinematic backgrounds and smooth micro-animations.
- **Responsive Navigation:** Sophisticated sticky header with a mobile-ready hamburger menu and transparent-to-solid transitions.
- **Optimized UI/UX:** Built with the **Outfit** font family, using a curated blue-teal color palette with lime-green accent highlights.

---

## 🛠 Technology Stack

- **Framework:** [Angular v21](https://angular.io/)
- **Styling:** Vanilla CSS3 (Custom Design System) & [Bootstrap 5](https://getbootstrap.com/)
- **Icons:** [Bootstrap Icons](https://icons.getbootstrap.com/) & [FontAwesome 6](https://fontawesome.com/)
- **Animations:** [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)
- **Components:** Modular Angular components for Header, Footer, Hero, and Page Sections.

## 💻 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Recommended: Latest LTS)
- [Angular CLI](https://angular.io/cli) installed globally (`npm install -g @angular/cli`)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Stalin-Kcs/annaiec.git
   cd annaiec
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Install Core Libraries:**
   ```bash
   npm install bootstrap aos bootstrap-icons
   npm install --save-dev @types/aos
   ```

4. **Install Firebase Packages:**
   ```bash
   npm install firebase
   npm install @angular/fire --force
   ```

### Development Server

Run `npm start` or `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
